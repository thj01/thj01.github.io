Joplin

Joplin er et opensource program, hvor man kan oprette notesbøger med noter og to-do lister. Fordelen ved at anvende Joplin fremfor andre er, at det virker på alle platforme, og at formatet er åbent og simpelt. Dvs. man kan senere konvertere det til andre formater.

Derudover findes Joplin også til alle de platforme man anvender

http://joplin.cozic.net/

Joplin kan endda anvendes fra kommandolinien - http://joplin.cozic.net/terminal/

Synkronisering

Joplin kan synkronisere med lokale netværksdrev og personlige Microsoft Onedrive konti, og der arbejdes på at få den til at synkronisere med Dropbox.

Markdown format

Man kan vælge enten at skrive helt almindelige noter, men der kan også laves avanceret layout ved hjælp af markdown format (https://daringfireball.net/projects/markdown/basics).

Overskrifter Markdown
Overskrift 1 (h1) #
Overskrift 2 (h2) ##
overskrift 3 (h3) ###
kursiv *tekst* eller _tekst_
Fed **tekst** eller **tekst**

Kode eksempel

	
# Overskrift 1

Dette er en almindelig tekst med **fed**

## overskrift 2

Her er det *kursiv*

### overskrift 3

Lister

Lister kan laves med tegnene +, - eller *. Uanset hvilket tegn du anvender vil der komme en liste med punkter.

	
Dette er en liste

* Første element
* Andet element
* Tredie element

Det kan også laves med tal

	
Dette er en liste

1 Første element
2 Andet element
3 Tredie element
