Binære tal

En effektiv måde at arbejde med data på er, at anvende binære tal. Binære tal er specielle ved, at de på hver position kun kan antage to værdier 0 og 1.

Lad os kigge på et eksempel der anvendes til at skriverettigheder på Linux:

Bruger Gruppe Alle
read write execute read write execute read write execute
r w x r - x r - x
1 1 1 1 0 1 1 0 1
7 5 5

Det som skemaet fortæller er, at der tildeles rettigheder til 3 forskellige sammenhænge: Brugeren, gruppen og alle. I skemaet er der tre rækker der hver især beskriver hvordan rettighederne er fordelt.

  • 1. række - Teksten er skrevet helt ud
  • 2. række - Rettighederne præsenteret ved hjælp af symboler
  • 3. række - Rettighederne præsenteret ved hjælp af et binært tal
  • 4. række - Rettighederne præsenteret ved hjælp af decimaltallet for hver af brugergrupperne: ejer, gruppe og alle.
