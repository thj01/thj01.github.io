Installation af Ubuntu 14.04.4 på Windows 7

Multiboot systemer

Det er muligt at have installeret mere end et operativsystem på en computer, når man har en boot-manager. Vi vil i denne sammenhæng installere Ubuntu på en Windows 7 computer, da det er det nemmeste. For at skabe plads til Ubuntu skal vi først have formindsket harddisken og det gør vi her ved hjælp af Windows' egen diskhåndtering.

Formindsk harddisken - Diskhåndtering

Du formindsker en disk ved at:

  1. Vælg start og kør kommandoen: compmgmt.msc
  2. Gå til: Computeradministration → Lager → Diskhåndtering
  3. Højreklik på den harddisk du vil formindske og vælg "Formindsk Diskenhed"
  4. Skriv hvor meget du vil formindske den og vælg "Formindsk"

    5. Du bør også tjekke om harddisken er uden fejl

  5. højreklik på den formindskede partition og vælg: Egenskaber → Fanen: Funktioner → "Undersøg nu..." og tryk "start".

    6. Har du fulgt disse trin er du klar til at installere Ubuntu.. Men tag alligevel backup af dine filer!

Installationssprog, kontrol af computer og netværk

Opsætning af harddisk

Computeren vi installerer på i dette eksempel har harddisken sat op på følgende måde.

Drev Betegnelse Størrelse Anvendelse
/dev/sda1 / 50000Mb (50Gb) System
/dev/sda2 Ingen 4096MB (4GB) Swap
/dev/sda3 /data 31000MB (31GB) Data og backup

I Windows' "Computeradministration " ser det ud som nedestående:

Vælg "Noget andet" og "Fortsæt.

Læg mærke til navngivningen af drevene:

Ubuntu kalder dem Windows kalder dem
/dev/sda1 - Disk 0 Partition 1 Reserveret til system
/dev/sda2 - Disk 0 Partition 2 C:
/dev/sda3 - Disk 0 Partition 3 D:

Vi vil herunder tilføje 2 partitioner. 1 til swap (/dev/sda5) og en til system (/dev/sda6). Til aller sidst vil vi monteret /dev/sda3 som /data så vi har adgang til backupfiler både fra windows og Ubuntu

Det som kræver tilvendning fra en Windows bruger er, at man er vant til at drev anvender drevbogstaver, hvor Ubuntu anvender navngivne monteringspinkter, som kan monteres efter behov - og hvor de giver mest mening.

Marker "ledig plads" og tryk på "+"

Monteringspunkt: /

Herefter skal vi have D-drevet (data/backup) fra Windows monteret ( Disk 0 Partition 3 - /dev/sda3). Det gøres ved at markere /dev/sda3 og klikke på "Change".

Størrelse: 32506 MB (Lad være med at ændre)
Brug som: ntfs
Formater partitionen: NEJ (ingen markering)
Monteringspunkt: /windows

Kontroller at:

Enhed Type Monteringspunkt Formater
/dev/sda3 ntfs /windows NEJ
/dev/sda5 swap NEJ
/dev/sda6 ext4 / (roden) JA

Er der fejl så gå tilbage og lav "change" indtil det er korrekt. Hvis du formaterer f.eks. /dev/sda2, som er systemdrev på Windows så er filerne væk. Så kontroller det grundigt.

Vælg: "Installer nu" når du er klar

Her bliver du informeret om at Ubuntu opretter og formaterer partitionerne /dev/sda5 og /dev/sda6 som henholdsvis swap og Ext4.

Herefter installerer Ubuntu sig færdig. Ubuntu henter opdateringer og lande/sprog konfigurationer på nettet

Vælg: "Genstart nu"

Til forskel fra Windows så beder Ubuntu dig om at fjerne boot-mediet. Det skyldes at Ubuntu ikke noget behov for at blive genstartet under installationen - og derfor heller ikke noget behov for at beholde boot-mediet monteret. Tast: [Enter]
