Windows 10 opsætning

Forbered Windows

Gå ind i programmet Computeradministration ved at klikke på forstørrelsesglasset og skriv compmgmt.msc

Højreklik på den partition der skal formindskes (her C:) og vælg Formindsk diskenhed.

Windows tjekker drevet for ledig plads.

Det næste vindue fortæller dig hvor stor din partition er og hvor meget du kan formindske den med. Mit råd er at du vælger mindst 20 GB, så du ikke løber tør for plads lige med det samme. Her vælger jeg 30GB ( 30000MB ).

På næste vindue kan du se resultatet. Ubuntu vil blive installeret på den ikke-allokerede del af harddisken.

Tjek harddisken

For at sikre os vælger vi at kontrollere den partition vi formindskede. Egentlig bør det ikke være nødvendigt, for Windows laver selv en hurtigt kontrol når vinduet Computeradministration opdateres. Vi gør det, fordi det er en ekstra sikkerhed for at Windows er OK.

Højreklik på den formindskede partition ( her C:) og vælg Egenskaber.

Klik på Funktioner og klik på Undersøg i feltet fejlsøgning.

Som skrevet ovenfor har windows lige kontrolleret drevet og gør opmærksom på at det ikke er nødvendigt. Vi gør det alligevel for det skader ikke noget.

Læn dig tilbage og lad Windows køre sin fejlsøgning af drevet.

Windows svarer til bage at der ikke er fundet nogle fejl.

Melder windows fejl her, så følg den vejledning Windows giver.
