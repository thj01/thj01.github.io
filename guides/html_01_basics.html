HTML

Et HTML5 dokument er grundlæggende opbygget efter nedesteånde skabelon

Opbygning af HTML

	
<!DOCTYPE html>

<html>

<head>

	<title>HTML Tutorial</title> so

	<meta charset="UTF-8" lang="da"/>
		
	<link rel="stylesheet" href="style.css" />

</head>

<body>

	<h1>Overskrift 1</h1>

	<p>Brødtekst</p>
	
</body>

</html>

Gennemgang

<!DOCTYPE html>

Denne linie fortæller browseren, at det er et HTML5 dokument og hvordan de forskellige tags skal anvendes.

<head>

I denne sektion sætter man alle de overordnede parametre for HTML-dokumentet. I det eksemplet herover bliver følgende parametre sat:

  • <title> - Siden titel
  • <meta> - tegn- og landekode
  • <link> - Den fil HTML-dokumentet skal hente designet af elemeterne fra
