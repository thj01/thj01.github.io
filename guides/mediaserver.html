Mediaserver

Installation

sudo apt-get install minidlna

Forberedelse

Opret mapper

sudo mkdir -p /media/MediaServer/Musik /media/MediaServer/Video /media/MediaServer/Billeder

Ændr bruger og gruppe

sudo chown -R minidlna:minidlna /media/MediaServer

Ændr rettigheder

sudo chmod -R775 /media/MediaServer

Tilføj din bruger til gruppen minidlna

sudo adduser tj minidlna

For at få ændringer til at slå igennem skal grupperne genindlæses

exec su -l $USER

Kopier dine data

Herefter kan du kopiere din musik, videoer og billeder til de nye mapper

Konfiguration

sudo nano /etc/minidlna.conf

Indsæt følgende i bunden af filen:

friendly_name=MediaServer
media_dir=A,/media/MediaServer/Musik
media_dir=V,/media/MediaServer/Video
media_dir=P,/media/MediaServer/Billeder
inotify=yes

Genstart minidlna

sudo service minidlna restart

Opdater databasen

sudo service minidlna force-reload

Enjoy

Tjek Mediaserveren

Du kan kontrollere om din mediaserver er oppe ved at åbne en browser og indtaste følgende:

localhost:8200
