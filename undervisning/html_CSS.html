CSS

Hvad er CSS

CSS står for Cascading Style Sheets, og er den metode man anvender til at formatere sine websider. Det fungerer ved at CSS koden fortæller browseren hvordan den skal vise HTML-koden.

I praksis gør man det ved at linke til en CSS-fil i HTML-sidens <head>-sektion. Det med rødt er min tilføjelse for at gøre det tydeligt.

	<!DOCTYPE html>
<html>
<head>
	<meta charset="UTF-8" lang="da"/>
	
	<link rel="stylesheet" href="style.css" /> <!-- Link til CSS-->
	
	<title>HTML skabelon med CSS</title>
	
	..

Herefter oprettes filen style.css i samme mappe som ens HTML-dokument, og CSS kommandoerne indsættes i style.css. For at ramme de forskellige tekstelementer med anvender vi deres tags. Som du så i ovenstående eksempel skriver vi indholdet af tag'et efterfulgt af curly-brackets ( { } ) og skriver CSS-koden deri. Jeg har indsat kommentarfelter i CSS kode ( tekst der er omgivet af /* */, for at beskrive hvad der sker.

Læg mærke til enheden em. Em angiver størrelsen i forhold til den font størrelse dokumentet anvender (16px som standard). 2em svarer til \( 2 \cdot 16 = 32 px \) og 1.5em til \( 1,5 \cdot 16 = 24px \). Vil man ændre størrelsen af skriften i hele dokumentet sætter man en universel standardstørrelse ved hjælp af * som herunder 

	/* Universel fontstørrelse (standard) */
* {
font-size: 16px;
}

/* Overordnet formatering der gælder alle overskrifttyper */	
h1, h2, h3 {	
font-family: arial;
font-weight: 900;
}

/* Formatering der gælder de enkelte overskrifttyper */
h1 {
font-size: 2.2em;
}

h2 {
font-size: 1.8em;
}

h3 {
font-size: 1.3em;
}

/* Formatering der gælder <p> */
p { 
font-family: verdana;   
}

Målrettet design

Som vi så ovenfor, så kan man anvende tags til at målrette sit design, men der opstår også andre situationer. Det kan man opnå med <div>, <span>, id="**ID_NAVN**" og class="**CLASS_NAVN**":

  • Man skal nogle gange anvende et lidt andet design på et design, en eller flere gange ( id og class)
  • Man skal ramme en gruppe af elementer ( <div> )
  • Man skal nogle gange formatere noget inde i et tag ( <span> )

id og class

id og class anvendes til at identificere enkelte elementer.

Herinder har jeg ændret lidt i skabelonen og indsat id="red" i den første <p> tag og class="blue"> ved de sidste to. Forskellen mellem id og class er, at id beskriver et konkret element og anvendes en og kun engang, hvormod class kan ramme mange elementer.

	<h1>Overskrift</h1>
	
<p id="red">Lorem ipsum dolor sit amet, consectetur adipiscing elit. Curabitur condimentum ac nisi a sagittis. Pellentesque fringilla quis turpis vitae congue. Integer vel eleifend ex. Quisque ultricies tempor sem a facilisis.</p>
	
<h2>Under-overskrift</h2>
	
<p class="blue">Duis eu quam a augue commodo lobortis. Nulla quis justo ac nulla venenatis finibus. Nam non vestibulum lectus. Nunc nec sapien scelerisque, egestas felis at, suscipit elit.</p>
	
<h3>Under-under-overskrift</h3>
	
<p class="blue">Lorem ipsum dolor sit amet, consectetur adipiscing elit. Curabitur condimentum ac nisi a sagittis. Pellentesque fringilla quis turpis vitae congue. Integer vel eleifend ex. .</p>

Derudover indsættes følgende linier i css-filen:

	p#red {
color: red;
}

p.blue {
color: blue;
}

<div> og <span>

Både <div> og <span> anvendes til gruppere indhold, men de fungere lidt forskelligt. <div> er et block-element, hvilket vil sige at browseren vil starte divsektionen i en ny linie.
