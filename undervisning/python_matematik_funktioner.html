Funktioner

Dette kan vi kombinere i en funktion, som skal udføre følgende kommandoerne lige efter hinanden ved at definere funktionen brøkregner(). Herunder laver vi funktionen brøkregner(). Det vi gør er reelt at samle det program der står ovenfor. Det er altså:

  • Indhente tælleren (brugerinput)
  • Indhente nævneren (brugerinput)
  • beregne og udskrive brøkens float værdi (output)

Når programmet tastes ind kommer det til at se ud som følger

	
from fractions import Fraction 

def brøkregner(a,b):
	floor= a//b
	tæller = (a-(floor*b))
	restbrøk = Fraction(tæller,b)
	print("Heltallet er {0} og brøken er {1}".format(floor,restbrøk)) 

brøkregner(45,37)
	

Heltallet er 1 og brøken er 8/37

Vi tager det linie for linie:

  1. def brøkregner():

    2. def fortæller at vi vil definere funktionen brøkregner(). Linien skal afsluttes med et : (kolon), for at vise at der høre noget kode ind under funktionen. Herefter skiftes der til næste linie hvor >>> erstattes af ..., som angiver roden af programmet.

  2. ...     a=int(input("Skriv tæller: "))

    3. Næste linie der henter tælleren er rykket ind under def brøkregner():. Det er måden man indlejrer kode under noget andet. Det gøres med Tabulator eller ved at bruge 4*Mellemrum

  3. ...     b=int(input("Skriv nævner: "))

    4. Er rykket ind ligesom ovenstående linie og henter nævneren

  4. ...     print(a/b)

    5. Udskriver resultatet af funktionen

Nu kan funktionen køres i pythonfortolkeren som enhver anden funktion:

>>> brøkregner()
Skriv tæller: 3
Skriv nævner: 7
0.42857142857142855
>>>

Smarte funktioner

Funktioner kan gøres rigtig smarte, så de ikke behøver brugerinput, men kan nøjes med at angive inputs i parantesen. Se nedestående eksempel:

>>> def nybrøkregner(a,b):
....	print("Brøken {0}/{1} giver {2}".format(a,b,a/b))
....
>>> nybrøkregner(3,4)
Brøken 3/4 giver 0.75

I selve funktionen defineres værdierne a og b. Disse to værdier hives ned i koden og giver et svar.
