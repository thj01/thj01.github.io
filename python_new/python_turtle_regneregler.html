Regneregler

Regningsarterne fungerer ved at anvende tegnene: +, -, * og /. Se herunder:

Regningsart Symbol i Python
Plus +
Minus -
Gange *
Division /

Herunder er det shell der er blevet brugt til at lave beregninger.

	>>> 6+3
9
>>> 6-3
3
>>> 6*3
18
>>> 6/3
2.0

Parenteser anvendes som man er vant til

	>>> 3+2*6
15
>>> (3+2)*6
30
>>> 3+6/2
6.0
>>> (3+6)/2
4.5

Pi (π)

Hvis du skal bruge pi er det lidt mere bøvlet, fordi vil skal importere matematikmodulet.

>>> import math
>>> math.pi
3.141592653589793
>>> 3 * math.pi
9.42477796076938
