Turtle

Et simpel turtle program

	from turtle import *

forward(100)
left(90)
forward(100)
left(90)
forward(100)
left(90)
forward(100)
left(90)

Resultat

Det koden siger er:

  1. Importer turtleprogrammet
  2. Bevæg cursoren 100 frem
  3. Drej cursoren 90° til venstre
  4. Bevæg cursoren 100 frem
  5. Drej cur...
  6. ...

De simple kommandoer er her:

Kommando Alternativer Resultat
forward() fd() Fremad - længden angives i parantesen
backward() bd() Tilbage - længden angives i parantesen
left() lt() Til venstre - vinklen angives i parantesen
right( *værdi*) rt() Til højre - vinklen angives i parantesen

Cirkelbuer

Lad os tegne en simpel cirkel

from turtle import *

circle(100)

Herunder kan du se hvad der sker.

Tallet i parantesen angiver afstanden til cirklens centrum vinkelret på retning markøren har.

Dele af cirkelbuen

Man kan tegne en mindre del af cirkelbuen ved at sætte et komma, og tilføje antallet af grader.

from turtle import *

circle(100,90)
 
from turtle import *

circle(100,180)
 
from turtle import *

circle(100,270)

Herunder er eksempel hvor jeg tegner en linie, med en cirkelbue på midten:

from turtle import *

forward(100)
right(90)
circle(50,180)
right(90)
forward(100)
