Introduktion til PHP

Hvad er PHP

PHP er et akronym for "PHP: Hypertext Preprocessor". Formålet med PHP er, at lave dynamisk indhold på hjemmesider, som både kan komme fra brugerinput og databaser. Vi skal seneer lære hvordan vi integrerer PHP-kode på hjemmesider.

I denne guide vil du lære at:

  • Afprøve og teste programmer ved hjælp af den indbyggede shell
  • Lave mindre programmer ved hjælp af Geany
  • Integrere PHP-funktionalitet i hjemmesider

PHP shell

PHP's shell startes op ved at at starte PHP op med -a

På Windows

c:\xampp\php\php -a

På Mac

/Applications/XAMPP/bin/php/php -a

Når man starter PHP shell op får man følgende resultat:

Interactive shell

php >       ← Her indtastes koden

Kommandoer indtastet efter php >

Mit første program

Lad os lave et simpelt program der udskriver tekstrengen "Hej" og tallet 4. Det gøres ved at anvende kommandoen echo

php > echo "Hej";       ← kommando
Hej                     ← output
php > echo 4;           ← kommando
4                       ← output
php >

Læg mærke til at der er forskel på hvordan tekststrengen "hej" og værdien 4 bliver indtastet. Reglen er, at alt hvad PHP skal opfatte som en værdi (dvs. tal) skal skrives uden anførselstegn. Tekst skal altid omsluttes af anførselstegn.

Det andet du skal lægge mærke til er, at alle kommandoer skal afsluttes med ;. På den måde ved PHP at en enkelte kommando er færdig. Det kan vi senere udnytte til at skabe bedre overblik over mere komplekse kommandoer.

Du skal altså huske:

  • Tekstrenge skal være omgivet af udråbstegn
  • Værdier skal ikke være omgivet af tekstrenge, og decimaltal skal adskilles af punktum (.) f.eks: 3.14

Escape characters

Når man arbejder i shell kan man nogle gange have brug for at skabe mere visuelt overblik. Her kan man anvende følgende escape characters:

  • \n - ny linie
  • \t - tabulator
php > echo "\nhej\n";

hej

php > echo "\n\thej";

        hej

Kommentarer

Når man arbejder med større programmer anvender man kommantarer til at forklare kodestumper. Her er vores ovenstående program med nogle forskellige kommentarer i


	// Et lille program

echo "Hej"; // en kommentar til denne linie

/*
Dette er en længere kommentar,
som kan fylde over flere linier
*/

Som du kan se kan vi indsætte kommentarer:

  • Ved at skrive // foran
  • Ved at begynde med /* og afslutte med */
