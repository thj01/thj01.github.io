Strenge og variabler

Hvad er PHP

PHP er et akronym for "PHP: Hypertext Preprocessor". Formålet med PHP er, at lave dynamisk indhold på hjemmesider, som både kan komme fra brugerinput og databaser. Vi skal seneer lære hvordan vi integrerer PHP-kode på hjemmesider.

Programmerne kan køres på to måder. Enten igennem en browser eller ved hjælp af en shell. I disse dokumenter vil selve introduktionen af grundfunktionerne blive vist igennem shell, og senere vil vi integrere PHP'en i rigtige HTML dokumenter.

PHP's shell startes op ved at at starte PHP op med -a

På Windows

c:\xampp\php\php -a

På Mac

PHP shell

Når man starter PHP shell op får man følgende resultat:

Interactive shell

php >

Kommandoer indtastet efter php >

Mit første program

Lad os lave et simpelt program der udskriver tekstrengen "Hej" og tallet 4. Det gøres ved at anvende kommandoen echo

php > echo "Hej";       ← kommando
Hej                     ← output
php > echo 4;           ← kommando
4                       ← output
php >

Læg mærke til at der er forskel på hvordan tekststrengen "hej" og værdien 4 bliver indtastet. Reglen er, at alt hvad PHP skal opfatte som en værdi (dvs. tal) skal skrives uden anførselstegn. Tekst skal altid omsluttes af anførselstegn.

Det andet du skal lægge mærke til er, at alle kommandoer skal afsluttes med ;. På den måde ved PHP at en enkelte kommando er færdig. Det kan vi senere udnytte til at skabe bedre overblik over mere komplekse kommandoer.

Du skal altså huske:

  • Tekstrenge skal være omgivet af udråbstegn
  • Værdier skal ikke være omgivet af tekstrenge, og decimaltal skal adskilles af punktum (.) f.eks: 3.14

Escape characters

Når man arbejder i shell kan man nogle gange have brug for at skabe mere visuelt overblik. Her kan man anvende følgende escape characters:

  • \n - ny linie
  • \t - tabulator
php > echo "\nhej\n";

hej

php > echo "\n\thej";

        hej

Kommentarer

Når man arbejder med større programmer anvender man kommantarer til at forklare kodestumper. Her er vores ovenstående program med nogle forskellige kommentarer i


	// Et lille program

echo "Hej"; // en kommentar til denne linie

/*
Dette er en længere kommentar,
som kan fylde over flere linier
*/

Som du kan se kan vi indsætte kommentarer:

  • Ved at skrive // foran
  • Ved at begynde med /* og afslutte med */

Variabler

Variabler er nemmere at vise end at forklare. Se nedestående eksempel.

php > $var = "hej"
php > echo $var;
hej

En variabel er noget information, som programmet kan anvende senere ved at henvise til den betegnelse vi har valgt. I dette tilfælde $var. Der gælder følgende for en variabel:

  • Alle variabler starter med $
  • Et variabelnavn må ikke starte med et tal.

Ændring af variabler

En af fordelene ved at anvende variabler er, at vi kan ændre den løbende. 

$var = "Hej";
echo $var;
$var = " med dig";
echo $var;

Det som sker er, at PHP udskriver variablen $var to gange. Første gang er den: "Hej" og anden gang " med dig". Læg mærke til at jeg har indsat et mellemrum før med. Ellers ville der komme til at stå: Hejmed dig

Et andet eksempel med tal kan være:

php > $antal = 10;
php > $pris = 9.95;
php > echo $antal * $pris;
99.5

Det vi kan anvende varibler til er bl.a. at styre input. Hvis vi f.eks. skal anvende en brugers navn, så kan vi oprette variablerne $f_navn og $e_navn. Vi kan altså skrive vores kode så den gælder for alle de navne brugerne kan indtaste.

Variabler i teksts

Ofte har vi brug for at udskrive variabler som en del af en tekst. Se eksemplet herunder

php > $navn = "Thomas";
php > $alder = 47;
php > echo "Mit navn er $navn og jeg er $alder gammel.";
Mit navn er Thomas og jeg er 47 gammel.

Opgave

Lav nedestående tekststykke ved hjælp af PHP hvor de ord der står mellem *'er skal angives som variabler.

	Mit navn er *navn*. Jeg bor i *by* og er *alder* år gammel. Mit telefonnummer er: 
*tlf_nr* og det er en *tlf_mrk*
