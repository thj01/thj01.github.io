Bootstrap - Intro

Bootstrap er et HTML-framework, som oprindeligt er blevet udviklet af Twitter. Fordelene ved bootstrap er:

Responsive design og mobile first
Simpel styring af design på forskellige enheder: telefoner, tablets, små og store skærme.
Modular opbygning
Sider opbygges ved at oprette fortløbende moduler. Kaldes containers i bootstrap.
Grundlæggende design af udseende
Bootstrap har nogle visuelle design, som kan bruges direkte og uden det store kendskab til CSS.
Open source
Bootstrap frit anvendeligt, og kan bruges uden forbehold.

Bootstrap - opbygning

Bootstrap sider har følgende opbygning:

<!DOCTYPE html>
<html lang="da">
<head>
  <title>Bootstrap Example</title>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
</head>
<body>

*** bootstrap indhold ***


</body>
</html>

Det kan være en fordel at anvende tjenester som maxcdn eller cloudflare til at hente CSS og JS filer, da mange brugere allerede har dem liggende på deres enheder - fordi bootstrap er så udbredt som det er.

Selve sidens indhold bygges op ved hjælp af containere

Containers

Bootstrap sider opbygges af <div< -elementer, der får tildelt klassen container.

Sidens indhold lægges i disse containers

<div class="container">

	*** Indhold af første container ***
	
</div>

<div class="container">

	*** Indhold af første container ***
	
</div>

<div class="container">

	*** Indhold af første container ***
	
</div>

Bootstrap har en indbygget Call to action -container der kaldes en jumbotron. Den skal altid lægges øverst på siden.

<div class="container">
  <div class="jumbotron">
    <h1>Overskrift</h1>      
    <p>Beskrivelse af sidens indhold</p>
  </div>
</div>

Fordelen ved at anvende jumbotrons er, at du kan designe sidehovedet, da jumbotron -klassen har en masse standardindstillinger.

Sidens bredde

Du kan designe containerens størrelse ved hjælp af klasser.

Class Extra small
<576px 		Small
≥576px 		Medium
≥768px 		Large
≥992px 		Extra large
≥1200px
.container-sm 100% 540px 720px 960px 1140px
.container-md 100% 100% 720px 960px 1140px
.container-lg 100% 100% 100% 960px 1140px
.container-xl 100% 100% 100% 100% 1140px
.container-fluid 100% 100% 100% 100% 100%

Hvis du laver en container med klassen container-md betyder det, at siden vil have 100% bredde på enheder med skærmstørrelse mindre en 992px. Er skærmen over 1200px, så vil maxstørrelse blive sat til 1140px

Sætter du skærmen til container-fluid, så vil hjemmesiden altid fylde 100% af skærmen, med mindre du har givet <body> elementet en max-width.

Containerens proportioner og udseende

Udover siden bredde, kan du også designe de enkelte containeres proportioner og udseende. Herunder er fra boks-modellen.

Grundliggende om proportioner og udseende

Bootstrap bruger en konstant måde at navngive sine klasser. Klasserne bliver gennemgået længere nede.

.w-XX
width (bredde) - kan være 25,50,75,100%, auto
h-XX
height (højde) - kan være 25,50,75,100%, auto
p-XX
padding - kan være 0, 1, 2, 3, 4, 5, som størrelse i fht. sidens standardskrift
m-XX
margin - kan være 0, 1, 2, 3, 4, 5
border
rammen - hvordan rammen skal se ud

width og height

width og height bliver sat i % af sidens bredde.

width:

w-25

w-50

w-75

w-100

w-auto

Se koden til width 
<p class="w-25 bg-success">w-25</p>
<p class="w-50 bg-success">w-50</p>
<p class="w-75 bg-success">w-75</p>
<p class="w-100 bg-success">w-100</p>
<p class="w-auto bg-success">w-auto</p>

height

Herunder kan du se hvad h-XX klassen gør. Rækken herunder er sat til 200px i højden. h-XX bruges så til at sætte højden på hver af rækkerne.

h100
h-75
h-50
h-25
Se koden til height 
<div class="row" style="height:200px" >

	<div class="col-3 h-100 border bg-light" >h100</div>
	<div class="col-3 h-75 border bg-light">h-75</div>
	<div class="col-3 h-50 border bg-light">h-50</div>
	<div class="col-3 h-25 border bg-light" >h-25</div>

</div>

Grid

Du kan designe opbygningen af siden på forskellige enheder, med Gridsystemet.

Opbygning af Bootstrap side

En af fordelene ved at arbejde med Bootstrap som framework er, at man hurtigt kan få et effektivt og responsive design af sine hjemmesider. Grunden til at dette kan lade sig gøre er, at der ligger en masse prædefinerede funktionalitet, sådan at man kan kan koncentrere sig om funktionalitet og brugervenlighed.

For at kunne arbejde effektivt med Bootstrap skal man have en forståelse for dets anvendelse af dets grid-system, &<div>-blokke og anvendelse af klasser. Det vil være i fokus i disse udvalgte guides fra w3schools.

Du skal nu arbejde med disse fire kapitler i denne touturial:

  • BS HOME: http://www.w3schools.com/bootstrap/default.asp
  • BS Get Started: http://www.w3schools.com/bootstrap/bootstrap_get_started.asp
  • BS Grid Basics: http://www.w3schools.com/bootstrap/bootstrap_grid_basic.asp
  • BS Typography: http://www.w3schools.com/bootstrap/bootstrap_typography.asp
  • BS Images: http://www.w3schools.com/bootstrap/bootstrap_images.asp
